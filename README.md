# Amazon Sales Data Analysis (Excel Project)

## Project Overview
This project analyzes Amazon sales data using Microsoft Excel. 
The dataset contains order information such as order ID, date, status, 
sales channel, shipping details, and product information.

The analysis focuses on extracting meaningful insights using Excel tools 
like Pivot Tables, filtering, sorting, and data cleaning techniques.

---

## Dataset Information
The dataset includes the following key fields:

- Order ID
- Date
- Status
- Fulfillment
- Sales Channel
- Ship Service Level
- Category
- Size
- Courier Status
- Quantity
- Amount
- Ship City
- Ship State
- Ship Country

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Filtering & Sorting
- Data Aggregation

---

## Analysis Performed

The following analysis was conducted using Pivot Tables:

1. Total sales by category
2. Order distribution by state
3. Sales trend by date
4. Quantity sold per product category
5. Order fulfillment analysis
6. Courier status analysis
7. B2B vs B2C order comparison

---

## Key Insights

- Certain product categories generated the highest revenue.
- Sales distribution varies significantly across different states.
- A large portion of orders were successfully delivered.
- B2C orders dominate compared to B2B orders.
